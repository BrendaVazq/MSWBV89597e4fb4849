3. Actualizaciones de código

4. Rediseño de algunas pantallas

5. Validaciones (correo, msm, whatsapp y llamada)

6. Actualizacón de versiones en Play Store / App Store

TRANSACT GATE


RESUMEN

TRANSACT GATE es una aplicación financiera desarrollada desde cero, diseñada para consultar datos de las tarjetas registradas en la empresa, así como sus movimientos bancarios.

Objetivos:

1. Diseño y estructura

3. Desarrollo de Base de Datos

4. Desarrollo e implementación de API's (se uso PHP)

⟿ Inició de sesión con usuario y contraseña asigando.

↣ Cambio de contraseña registrada por el usuario al momento de registrarse en la app.

⟿ Mostrar las trajetas registradas por el usuario

↣ El usuario pueda ver los movimientos de cada tarjeta registrada

⟿ Muestra el saldo disponible en cada tarjeta.

↣ Muestra un balance total del saldo disponible.

⟿ Identificar el tipo de cambio en la cuenta del usuario (Euro o Dolar).

TRANSACT GATE ADMIN


Objetivos:

1. El administrador obtiene los datos de la tarjeta ingresada; como el nombre, alias de la tarjeta, saldo disponible, número de tarjeta y tipo de cambio.

2. La aplicación permnite visualizar los movimientos y llevar el control de datos.
